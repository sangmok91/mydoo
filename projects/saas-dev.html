SaaS 개발·사업화 지원 사업

SaaS 솔루션의 개발부터 사업화까지 — 클라우드 네이티브 제품 개발을 지원합니다

📋 과학기술정보통신부 / NIPA 📅 공고 확인 필요 💰 최대 3억원

사업 개요

SaaS 솔루션을 개발하거나 기존 솔루션을 SaaS로 전환하여 사업화하려는 기업을 지원하는 사업입니다. 링키솔루션은 공급기업으로서 본 사업에 참여합니다.

핵심: SaaS 제품의 개발, 고도화, 시장 진출에 필요한 비용을 정부가 지원합니다.

지원 금액최대 3억원
지원 범위SaaS 개발, 클라우드 인프라, 보안인증, 마케팅
모집 기간공고 확인 필요
사업 기간최대 12개월

링키솔루션 개발 계획

기존 미니ERP 플랫폼을 고도화하여 SaaS 제품으로 사업화합니다.

개발 목표

  • AI 회계·세무 자동화 SaaS — 금융데이터 자동 수집 + AI 분류 + 장부·신고서 자동 생성
  • 멀티테넌트 고도화 — 기업별 독립 데이터 관리, 확장성 확보
  • API 오픈 — 외부 시스템 연동 API 제공
  • 모바일 앱 — iOS/Android 네이티브 앱

사업화 계획

  • 구독 모델 — 월 구독형 SaaS (기본/프로/엔터프라이즈)
  • 타겟 시장 — 중소기업 회계·경영관리 시장
  • 차별점 — 금융 스크래핑 + AI 자동분류 (경쟁사 없는 영역)
  • 보안인증 — CSAP 또는 SaaS 인증 취득
